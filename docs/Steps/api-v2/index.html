@qavajs/steps-api v2

Step library to work with web API

It is docs for qavajs v2. If you are looking docs for v1 go to (v1 docs)

Installation​

npm install @qavajs/steps-api

Configuration​

export default {
    require: [
        'node_modules/@qavajs/steps-api/index.js'
    ]
}

Parameter Types​

gqlRequestProperty​

Used to initialize one of two possible GraphQl body properties

  • query
  • variables

bodyParsingType​

Used for parsing responses body

  • arrayBuffer
  • formData
  • blob
  • json
  • text
